# Password-Analyzer
Password Analyzer is a cybersecurity project developed using Python, HTML, and CSS that checks password strength based on length, uppercase and lowercase letters, numbers, and special characters. It helps users create stronger and more secure passwords to improve online account safety.

## 📌 About The Project

**Password Strength Analyzer** is a cyber security project developed during my **Python Development Internship at Pinnacle Labs**.

This project helps users analyze password strength by checking important security conditions like:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Based on these checks, the system displays whether the password is:

✅ Strong Password  
⚠️ Medium Password  
❌ Weak Password

The project is designed with a modern cyber-security style interface using HTML and CSS while Python handles password analysis logic.
